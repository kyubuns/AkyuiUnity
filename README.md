# AkyuiUnity

AkyuiUnity is a Unity implementation of [Akyui](https://github.com/kyubuns/Akyui).  
AKyuiUnity.Xd, you can easily generate Unity's Prefab from [Adobe XD](https://www.adobe.com/products/xd.html).
